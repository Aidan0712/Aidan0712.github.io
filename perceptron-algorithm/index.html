💡 三、问题回答

1. 感知器算法的实现步骤

  1. 初始化：设置训练样本矩阵、类别标签、初始增广权向量以及修正系数。
  2. 增广处理：在每个样本后添加常数项1，将偏置项统一并入权向量计算。
  3. 遍历分类：依次遍历所有训练样本，利用当前权向量计算分类结果，判断样本是否被正确分类。
  4. 权向量修正：当样本被误分类时，按照感知器学习规则更新权向量，并记录更新次数。
  5. 迭代终止：完成一轮遍历后，若仍存在误分类样本则继续迭代；若所有样本均被正确分类，则算法停止。

2. 修正系数c对权向量的影响分析

通过对比c=1和c=0.5时的实验结果可以发现：当修正系数减小为原来的一半后，最终得到的增广权向量各个分量也恰好缩小为原来的一半。

结论：修正系数直接影响感知器算法中权向量每次更新的步长大小。较大的修正系数会使权向量更新幅度更大，最终权向量数值也更大；较小的修正系数则使权向量更新更加平缓，最终权向量整体数值相对较小。

2. PA子函数代码

%====================== PA子函数 ======================%
% X        : 训练样本矩阵
% w0       : 初始增广权向量
% c        : 修正系数
% classes  : 样本类别
function [w,k] = PA(X,w0,c,classes)
    [m, n] = size(X);
    % 增广样本矩阵
    Y = [X ones(m,1)];
    % 初始化权向量
    w = w0;
    % 迭代次数
    k = 0;
    while 1
        flag = 1;
        for i = 1:m
            % 判断当前样本是否被误分类
            if classes(i) * (w * Y(i,:)') <= 0
                % 更新权向量
                w = w + c * classes(i) * Y(i,:);
                % 更新次数加1
                k = k + 1;
                flag = 0;
            end
        end
        % 若全部分类正确则停止
        if flag == 1
            break;
        end
    end
end
