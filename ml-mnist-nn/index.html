2. 神经网络构建

扩展 torch.nn.Module，构建三层全连接神经网络： 

class MyNeuralNet(nn.Module):
    def __init__(self):
        super(MyNeuralNet, self).__init__()
        self.flatten = nn.Flatten()
        self.fc1 = nn.Linear(784, 256)
        self.fc2 = nn.Linear(256, 128)
        self.fc3 = nn.Linear(128, 10)
        self.relu = nn.ReLU()
    
    def forward(self, x):
        y = self.flatten(x)
        y = self.relu(self.fc1(y))
        y = self.relu(self.fc2(y))
        y = self.fc3(y)
        return y

💡 三、探究与思考

1. 预处理代码的功能

功能：对 MNIST 图像数据进行预处理，先转换为张量格式，再进行标准化。

两个 0.5 的含义：

  • 第一个 0.5 是均值
  • 第二个 0.5 是标准差

根据标准化公式 (x - mean) / std，将像素值从 [0,1] 映射到 [-1,1] 区间，有助于模型更快收敛。

2. DataLoader 的作用

DataLoader 将数据集包装成可迭代对象，实现批量加载数据、自动打乱顺序以及多进程并行读取，提高训练效率并方便对数据进行批次处理。

3. 输入层和输出层的神经元数量

  • 输入层：784 个神经元 — MNIST 图像大小为 28×28=784 像素，每个像素作为输入特征
  • 输出层：10 个神经元 — 手写数字识别是 10 分类问题（数字 0~9），每个神经元对应一个类别的输出概率

4. 损失函数的选择

本实验使用 交叉熵损失（CrossEntropyLoss）。

原因：本实验是多分类任务，交叉熵损失能衡量预测概率分布与真实标签之间的差异，配合 softmax 可有效优化分类模型。

5. 误差逆传播代码解析

optimizer.zero_grad()  # 清除之前迭代中保留的梯度参数，防止梯度累加
loss.backward()         # 根据当前损失计算网络中每个可训练参数的梯度
optimizer.step()       # 优化器根据计算出的梯度更新模型参数

6. 预测正确数量的代码解析

(pred.argmax(1) == y).type(torch.float).sum().item()

整体功能：统计当前批次中预测正确的样本数量

  • pred.argmax(1) — 对模型输出在类别维度取最大值的索引，得到预测类别标签
  • == y — 与真实标签进行比较，输出布尔值
  • .type(torch.float) — 将布尔值转为浮点数 1 和 0
  • .sum() — 求和，得到正确预测的数量
  • .item() — 将该张量转换为 Python 基本量
